## How to use CodeMap?
<br>

\(暂时只支持.swift文件\)

1、下载脚本

2、更改CodeMap.py中的目录地址

3、运行脚本

4、使用XMind\([https://www.xmind.cn/](https://www.xmind.cn/)\)导入生成的CodeMap.md文件



<br>

<br>

## What else can CodeMap do \(with XMind\)?
<br>

1、标注代码之间的关系

2、备注某段代码

3、把某个子树扩展到整个画布上



<br>

<br>

## How does CodeMap do it?
<br>

change code to .md to mind map

把代码转换为md文件再导入思维导图
